### 인턴

### 배포 url

- [http://karte.map.navercorp.com/](http://karte.map.navercorp.com/)

### Run

> ./start-karte.sh

```
cd karte
yarn
yarn start
```
- Style.json 파일 / icon 이미지 등은 저작권 문제로 소스코드에 포함하지 않는다.
